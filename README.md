1. СоцСеть пеших прогулок
Суть: Пользователи записывают свои пешие маршруты по городу и делятся ими: «Маршрут по дворикам Петербурга на 40 мин», «Прогулка с видом на закат в Казани». К маршруту прикладываются фото, аудио-комментарии, точки интереса. AI предлагает маршруты по настроению, времени и погоде.

Монетизация: авторские платные маршруты с комиссией + подписка для доступа к премиум-маршрутам + реклама кафе и заведений на маршрутах.

ЦА: любители прогулок, туристы, жители крупных городов.
