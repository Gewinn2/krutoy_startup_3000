## 2. Универсальное средство бесперебойного доступа к интернет-ресурсам.

### Проблема
Провайдеры по требованию регуляторов могут ограничивать доступ к целым пулам ресурсов (мессенджеры, соцсети, новостные сайты). Но даже если провайдер работает, у него могут возникнуть проблемы на транзитных линиях, из-за чего конкретный ресурс становится недоступен, например, пакеты теряются или не приходят. Пользователь не хочет вручную искать работающее подключение, постоянно переключаясь и проверяя результат.

### Решение
Система, которая работает по принципу умной динамической маршрутизации.

**Принцип работы**
1) Устройство подключается одновременно ко всем доступным каналам связи.
2) Система постоянно пингует именно тот ресурс, который на данный момент использует пользователь.
3) Как только система видит, что пакеты перестали доходить до целевого ресурса через текущего провайдера или скорость упала ниже установленного порога:
*    Она разрывает соединение с этим провайдером для данного трафика.
*    Автоматически переключает поток данных на другого доступного оператора.
*    Пробует подключиться заново (или продолжает сессию, если протокол позволяет).

Для пользователя интернет просто всегда работает. Все сложности с выбором SIM-карт и VPN-протоколов берет на себя софт.

### Конкуренты
*   **Обычные VPN-сервисы** Решают проблему доступа к определенному ресурсу, но не могут регулировать доступность всех ресурсов. Например, при включённом VPN будет работать Ютуб, но не будут работать Госуслуги.
*   **Роутеры с функцией 3G/4G резервирования (TP-Link, MikroTik)** Умеют переключать интернет с кабеля на 4G, если полностью пропал пинг. Однако они не анализируют доступность конкретного сайта.

### Расходы
*   Разработка MVP: Оплата труда бэкенд, фронтенд-разработчиков - около 350 т. р. на первые 3-4 месяца.

*   Инфраструктура (Сервера маршрутизации): Нам нужны свои серверы в разных дата-центрах (EU, USA, Asia), которые будут выступать точками сбора трафика и ретрансляции, чтобы пакеты не терялись при смене провайдера внутри страны (около 500$ в месяц на старте).

*   Закупка оборудования для тестов: Модемы, роутеры, SIM-карты разных операторов для отладки системы (около 100 000 руб.).

*   Юридические расходы: Оформление ООО, патентование полезной модели/алгоритма (50 000 - 100 000 руб.).


### Монетизация
**Подписка:**

Бесплатный тариф (Пробный):

*    Ограниченное количество переключений в день.

*    Только ручной выбор канала.

*    Ограниченная скорость

*    Реклама наших премиум-тарифов.

Премиум-подписка (Персональная):

*    Автоматическое «умное» переключение без ограничений.

*    Возможность подключения до 3 внешних модемов/SIM-карт одновременно.

*    Приоритетный доступ к быстрым серверам ретрансляции.

Цена: 500 руб/мес.

Корпоративный тариф (B2B):

*    Продажа готового "Железного ключа" (специальный роутер-коробочка), который сотрудники компаний подключают к своему компьютеру. Он имеет внутри слоты для 2-3 симок и настроенный софт.

Оплата за устройство + ежемесячная плата за обслуживание каналов.
